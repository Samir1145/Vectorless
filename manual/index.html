What is PageIndex?

PageIndex is a reasoning-based document intelligence platform. Its core feature — building hierarchical tree indexes from PDFs — is now extended with an Adversarial Multi-Agent Synthesis Pipeline specifically designed for legal adjudication.

The pipeline takes two opposing legal documents (Petitioner and Respondent), passes them through three AI agents, and produces a structured draft court order with full IRAC reasoning on every contested issue.

The Three Agents

🗂️
Clerk Agent
Reads each party's document and extracts structured facts, legal issues raised, statutory citations, and prayers for relief. Runs in parallel for both parties.
⚖️
Registrar Agent
Compares both clerk outputs side-by-side and frames every contested issue neutrally — mapping each party's arguments and citations against the same issue.
🔨
Judge Agent
Applies IRAC (Issue–Rule–Application–Conclusion) reasoning to each framed issue in sequence, then synthesises a final court order.

The Human Review Gate

Between the Registrar and the Judge there is a mandatory human review gate. No AI analysis proceeds until a human reviewer has examined the Adversarial Matrix and clicked Approve.

⚠️
Why this matters: The Adversarial Matrix determines what issues the Judge will decide and how each party's stance is characterised. A poorly framed matrix leads to a poorly reasoned order. Always review carefully before approving.

Quick Start (5 steps)

1
Upload both documents
Use the Documents panel to upload the Petitioner's and Respondent's PDFs. Wait for PageIndex processing to complete (status turns green).
2
Create a Case
In the Cases section, click + and give the case a title (e.g. "Ramesh Kumar v. MCD"). Then go to Setup, attach one document as Petitioner and one as Respondent.
3
Run the Clerk Agent
Click ▶ Run Clerk Agent. Both documents are processed in parallel. Switch to the Clerk tab to review extracted facts, issues, citations, and prayers.
4
Review and approve the Adversarial Matrix
Click ▶ Run Registrar Agent from the Clerk tab. Then go to the Registrar tab, review every framed issue, and click ✓ Approve.
5
Run the Judge and export
Click ▶ Run Judge Agent. Navigate to the Judge tab to read the IRAC analysis and final order. Export as Markdown with the download button.
